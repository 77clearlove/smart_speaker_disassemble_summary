# 智能音箱拆解总结

* 最新版本：`v0.8`
* 更新时间：`20190326`

## 简介

以智能音箱天猫精灵方糖为例，介绍如何拆解智能音箱的硬件，以及如何从硬件电路板中分析找出各种芯片，以及每种芯片的相信信息，并整理天猫精灵整个系列产品的拆解信息，常见智能音箱的的各种硬件信息，以及相关硬件背景基础知识和和拆解评测相关的工具。
